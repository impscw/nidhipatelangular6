• This application is built using Angular 6,Angular material,rxjs .
  • Home Component welcomes user to the website LetzEnterTain.
  • About Component describes functionality of the appliaction and information about developer.
  • Book Component allows user to see books offered by the website and all book images redirects to the book discription link.
  • Movie Component allows user to see movies offered by the websites.
  • Music Component allows user to listen to few artist song and link of songs redirects to music video.
  • All components are routed using navigation component of application.
